# devto-articles

This repository manages my Dev.to articles with GitHub.

## Structure

- `articles/drafts/`: draft articles
- `articles/published/`: published articles
- `templates/`: article templates
- `scripts/`: helper scripts
- `config/`: article mapping data

## Goal

- Write articles in Markdown
- Manage content with GitHub
- Publish/update posts to Dev.to with GitHub Actions
